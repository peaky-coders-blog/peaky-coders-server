# Start project

0. First steps

- `yarn prisma db push`
- `yarn prisma:seed`

1. Database
   `docker-compose up`

2. Server

- Client - `yarn start:dev client` || `yarn start:dev`
- Admin - `yarn start:dev admin`

3. Visual Database
   `yarn prisma studio`
